# Food Ordering Website Assignment

## Project Purpose
This repository contains a simple static food ordering website prepared for an SVU Web Design and Development assignment. The project demonstrates multi-page navigation, meal selection, detail toggling, form validation, and order summary calculation.

## Files
- `home.html`
- `AboutUs.html`
- `Meals.html`
- `Style.css`
- `myFunctions.js`
- `README.md`

## How to Run
1. Download or clone the project.
2. Open `home.html` in any modern browser.
3. Navigate through the pages and test the ordering flow from `Meals.html`.

## Technologies Used
- HTML
- CSS
- JavaScript
- jQuery

No backend, database, npm, or external build tools are used.
